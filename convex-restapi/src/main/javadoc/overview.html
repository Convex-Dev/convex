Convex REST API

HTTP REST server exposing Convex peer functionality over a conventional web API. Suitable for browser clients, language bindings without a native binary client, and integration with standard web tooling.

Endpoints include transaction submission, query execution, account lookups, and block / state inspection. The REST surface is public by default and designed to be robust against malicious messages.

Depends on convex-peer (and transitively convex-core).

Key Entry Points

  • convex.restapi.RESTServer — embeddable REST server
  • convex.restapi.api — endpoint handlers

See the Convex documentation for the full REST API reference.
