Convex DB

SQL database layer over Convex lattice data. Provides SQL query capabilities through Apache Calcite, a JDBC driver, and a PostgreSQL wire-protocol server — so standard clients such as psql, DBeaver, DataGrip, and Metabase can query lattice-backed tables directly.

Key Entry Points

  • convex.db.ConvexDB — root container for named SQL databases
  • convex.db.SQLDatabase — lattice-backed SQL database
  • convex.db.jdbc — JDBC driver
  • convex.db.pg — PostgreSQL wire-protocol server

Data is stored as convergent lattice structures, so replicas can be merged deterministically without coordination. See docs.convex.world for the full guide.
