Convex DLFS Server

Server-side tooling for the Decentralised Lattice File System (DLFS) — WebDAV endpoint, MCP tool integration, and drive management. Pairs with the DLFS NIO provider in convex.lattice.fs (part of convex-core) to expose lattice-backed file systems to standard file clients.

Key Entry Points

  • convex.dlfs.DLFSServer — embeddable WebDAV + management server
  • convex.dlfs.DLFSWebDAV — WebDAV protocol handler
  • convex.dlfs.DLFSDriveManager — manages named drives
  • convex.dlfs.DlfsMcpTools — Model Context Protocol tool bindings
