Convex Social

Social networking primitives built on Convex lattice data structures — users, posts, feeds, follows, and timelines with convergent merge semantics so replicas reconcile deterministically.

Key Types

  • convex.social.SocialUser — user identity
  • convex.social.SocialPost — a single post
  • convex.social.Feed, Follows, TimelineEntry — aggregation primitives
  • convex.social.SocialLattice — top-level lattice composition
