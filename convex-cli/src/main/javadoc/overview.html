Convex CLI

Command-line tooling for Convex — key management, transaction submission, peer operations, and scripting. Ships as part of the convex.jar uberjar produced by convex-integration.

Programmatic access to CLI commands is provided through this artifact for embedding in other tools and test harnesses.

Typical Entry Points

  • convex.cli.Main — CLI entry point (PicoCLI)
  • Subcommands for key, peer, transact, query, local, and desktop

See the Convex documentation for a complete command reference.
