Convex Peer

Peer-side implementation of the Convex network: the long-running server process that participates in Convergent Proof of Stake consensus, maintains global state, and exposes the binary Convex protocol to clients and other peers.

This artifact depends on convex-core and adds the networking, belief propagation, and block production layer required to run a node.

Key Entry Points

  • convex.peer.Server — the main peer server
  • convex.peer.Peer — in-process peer state and consensus loop
  • convex.net — Netty-based binary protocol

See the Convex documentation for operating peers and connecting clients.
