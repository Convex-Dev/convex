Convex Java Client

Java client library for connecting to Convex peers. Provides a synchronous and asynchronous API for submitting signed transactions, running read-only queries, and observing network state over the binary Convex protocol.

Depends on convex-core for the shared CVM data model.

Typical Usage

  • convex.java.Convex — high-level client facade
  • Sign transactions with an AKeyPair and an Address (both from convex-core)
  • Read results as immutable ACell values

See also: Java Client SDK guide.
